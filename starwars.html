Star Wars Info App

A personal project where I taught myself the basics of API integration with retrofit

About the Project

In this personal project, I practiced key Android and mobile app development skills by building an app that sends a GET request to the Star Wars API (SWAPI) and retrieves information about several characters from the Star Wars universe. The project integrates various Android development features to create a functional and interactive mobile application. I focused on API integration to fetch and display data from the SWAPI, utilizing coroutines to handle tasks such as data fetching and parsing to enhance app efficiency. Additionally, I implemented dark mode to improve the user experience and used view binding to simplify UI interactions. The app also features RecyclerViews to efficiently display lists of data.

Home Screen

In the home screen a get request to the SWAPI (Star Wars API) provides a list of star wars characters objects that fill the RecyclerView shown in the image. Displayed in this image is the list of characters available in the SWAPI API, the user may press on any character they wish and will be take to another screen that shows more information about each character.
