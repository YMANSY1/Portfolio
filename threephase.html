Three Phase Calculator App

A simple calculator for three phase systems that can find various parameters

About the Project

This app was a way to make calculating three phase system equations very simple and straightforward. It provides a simple interface made in Python's Tkinter GUI library that prompts the user to put some parameters about the current three phase system he wants to solve and pressing

Main Page

When you first open the app you are presented with the home screen that includes a list of parameters to fill so you could get the needed answer. Firstly, the user has to pick the desired circuit configuration from Star-Delta, Delta-Star, Star-Star, and Delta-Delta. The parameters needed are the magnitude and phase of the line voltage, the load impedance, the length and radius of the transmission line, the frequency, and the inner and outer layer of the conductor.

Results Page

Next is the Results page, where all the magic happens! All the calculations done will be displayed to the user in the shown window. The displayed results are: the phase voltages, the phase currents, the line currents, and the power. All the calculations were done using the math and cmath libraries in Python and the app was tested using questions from our textbook with 100% accuracy with regards to question solutions. This app was a good way for me to apply my university curricula into an actual project and one of the first applications I ever coded. This made developing the app truly exhilarating and makes it very dear to my heart despite it being very simple.
