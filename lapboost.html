Lapboost

A project for managing interactions with a laptop repair startup for both users and management using an integrated MySQL database

About the Project

This project allows the founders of the startup LapBoost to expand their database management from excel to SQL with a cross-platform application to boot! The app interacts with the database to allow users to make order from the user app and track said orders and services. The app includes an authentication system for users allowing for more efficient user management. The admin app is a work in progress but it will allow management of employees, orders, inventory, etc. Both apps allow the users to make all CRUD operations to the database. Here's a sneak peek of the user app!

Login and Signup Screen

A screen allowing users to make an account or login to their existing accounts.

Service Requests

After the user logs in they may request a service offered by LapBoost, the user simply chooses the service and saves their address if needed. The service is then recorded upon confirmation and will be sent to LapBoost management.

Marketplace

The users may also check the lapboost marketplace and buy all the components they need. they are given the ability to search for the component they need, or filter based on what they want to see. Once the user finds what they wish to buy, they will press on the item they desire and be sent to the screen to buy it.

Buying Items

The user may fill the information and select the quantity they need using the quantity counter, then they can press the confirm button and they order will be recorded. Providing an easy way to make orders and track them.

Order Tracking

The user app allows the users to track their orders, their price, what stage of delivery and processing they are in, and finally cancel the orders if needed. This allows for an all-in-one location for the users to interact with their LapBoost transactions.

Edit your account

The user app provides user with the ability to change all their information. Allowing them to update, add to, or delete their accounts all through the application. Making the interactions with the apps easier and more streamlined.

A Work in Progress

Since the project is still not complete and work on the admin app is still not yet finalised, the app cannot yet be displayed on this porfolio properly; however, this project has been a useful way of interacting with MySQL databases and using them to make an extremely useful application for both management and users alike!
