Saffron and Spice Meal Review App - IN PROGRESS

A Native Android App that allows restaurant managers to get user feedback seamlessly

About the Project

Saffron and Spice restaurant app offers a seamless experience for both customers and admins. Users can browse detailed menus, complete with item descriptions, prices, and images, as well as submit reviews, rate meals, and search for specific dishes quickly. Admins gain access to detailed reports on meal reviews and comments effortlessly. This app makes use of Supabase as the backend with a table containing all menu items and all relevant information about them. Java threads are used to manage asynchronous operations, making for a seamless and quick user experience.

Home Screen

When you first open the app you are presented with the home screen that includes two buttons, one will take you to the user screen and another to the admin screen.

Menu Page

Next is the menu screen; in this screen the user (or admin) may choose a meal they wish to access. A slightly modified version of this screen exists for both admins and users. The user screen includes a done button at the bottom that, when clicked after reviewing, displays a personalised message to user depending on the nature of their ratings. Three possible messages exist, one being elated about getting a perfect 5 star on all meals, the second thanking the user for positive feedback, and the third apologizing for a bad experience and assuring that their feedback made its way to upper management. Meal Review Screen

When the user clicks the meal they wish to review they are sent to the screen seen here. The screen allows the user to review individual ingredients and leave a comment. The user has to review at least one ingredient for the review to be recorded and the submit button to be clickable, leaving a comment is entirely optional. This screen serves as a simple and user-friendly way for the user to leave a review for the meals they ate.

Meal Report Screen

After the user presses submit on a review, their numbers are added to the average rating for each ingredient stored in the Supabase table. This average calculation is then retrieved and displayed in meal report screen for each meal. In this screen the number of stars denotes the average amounts of stars this meal recieved from users, and below it are the comments left by users about the meal. This provides clear and concise reporting about each meal, allowing upper management to derive insights from the displayed data. This project was amazing and fun learning experience for me. Learning Supabase and PostgreSql databases was a valuable learning experience and the use of Recycler Views and Java threads taught me new coding methods and tricks that have drastically increased my skills as an Android developer.
