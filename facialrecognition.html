Facial Recognizer for Healthcare

An award winning project that accurately detects the face of hospital staff and patients

About the Project

The Facial Recognizer for Healthcare project leverages Python's OpenCV library, utilizing the Viola-Jones algorithm for facial detection and Haar-like features for recognition. This award-winning initiative harnesses the power of machine learning and image processing to deliver precise and efficient facial recognition. Recognized for its innovation and impact, this project secured first place in the ECEN305 course rankings at the Undergraduate Research Forum. This accomplishment underscores the blend of academic rigor and real-world application, making it a standout feature in my portfolio.

GUI

When you first open the app you are presented with a screen where you can type in the name of a new patient or hospital staff member or immediately go on to detecting faces. After typing in the name of a new user, you may create and train a dataset for the new user and save their information in an integrated MySQL database where it can accessed anywhere and anytime by the hospital, making the processing seamless and efficient.

MySQL Database

The integrated MySql database provides an efficient and secure way to access the information about staff and patients registered on the system. The system cross-references the information in the database with the stored dataset which allows it to accurately recognize who is standing in front of the camera.

The Detection

When the user presses the 'Detect Face' button in the GUI, it takes them to a screen where the camera is used and the algorithm is applied to detect the face of the person staring at the camera. The system was rigorously tested, tallying up the true positives, false positives, true negatives, and false negatives; the results of the testing led to an impressive over 90% accuracy! This project is really dear to my heart as it is the first times one of my projects won an award. Standing up on the stage and recieving the award was an experience I will never forget, which made it one of the dearest projects in my portfolio.
